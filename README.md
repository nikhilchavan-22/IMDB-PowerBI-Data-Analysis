# IMDB Movie Data Analysis Dashboard
IMDB Movie Data Analysis Dashboard using Power BI.

## Project Overview
This project analyzes IMDB movie data using Power BI to identify trends in movie ratings, revenue, runtime, and popularity.

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Insights
• Genre-wise movie rating analysis  
• Revenue distribution across movies  
• Runtime vs rating comparison  
• Year-wise movie trends  

## Dataset
IMDB movie dataset used for visualization and analysis.
